# Развертывание приложения в интернете

## 1. GitHub Pages (Рекомендуется - Бесплатно)

### Шаги:

1. **Создайте репозиторий на GitHub:**
   - Перейдите на https://github.com
   - Нажмите "New repository"
   - Назовите репозиторий `DND` (или любое другое имя)
   - Сделайте репозиторий публичным
   - НЕ добавляйте README, .gitignore или лицензию (у вас уже есть)

2. **Загрузите код в GitHub:**
   ```bash
   # Инициализируйте Git (если еще не сделано)
   git init
   
   # Добавьте все файлы
   git add .
   
   # Сделайте первый коммит
   git commit -m "Initial commit"
   
   # Добавьте удаленный репозиторий (замените YOUR_USERNAME на ваш GitHub username)
   git remote add origin https://github.com/YOUR_USERNAME/DND.git
   
   # Переименуйте ветку в main
   git branch -M main
   
   # Загрузите код
   git push -u origin main
   ```

3. **Включите GitHub Pages:**
   - Перейдите в Settings вашего репозитория
   - Прокрутите до раздела "Pages"
   - В "Source" выберите "GitHub Actions"
   - Сохраните настройки

4. **Автоматический деплой:**
   - При каждом push в ветку `main` приложение автоматически соберется и задеплоится
   - Ваше приложение будет доступно по адресу: `https://YOUR_USERNAME.github.io/DND/`

## 2. Vercel (Альтернатива - Очень просто)

### Шаги:

1. **Установите Vercel CLI:**
   ```bash
   npm install -g vercel
   ```

2. **Деплой:**
   ```bash
   # В корне проекта
   vercel
   
   # Следуйте инструкциям в терминале
   # Vercel автоматически определит, что это Vite проект
   ```

3. **Автоматический деплой:**
   - Подключите GitHub репозиторий к Vercel
   - При каждом push приложение будет автоматически обновляться

## 3. Netlify (Альтернатива)

### Шаги:

1. **Соберите проект:**
   ```bash
   npm run build
   ```

2. **Загрузите на Netlify:**
   - Перейдите на https://netlify.com
   - Перетащите папку `dist` в область "Deploy manually"
   - Или подключите GitHub репозиторий для автоматического деплоя

## 4. Heroku (Для полноценного приложения)

### Создайте файлы для Heroku:

1. **Procfile:**
   ```
   web: npm run start
   ```

2. **package.json scripts (добавьте):**
   ```json
   "scripts": {
     "start": "vite preview --host 0.0.0.0 --port $PORT"
   }
   ```

3. **Деплой:**
   ```bash
   # Установите Heroku CLI
   # Создайте приложение
   heroku create your-app-name
   
   # Деплой
   git push heroku main
   ```

## Рекомендации

- **Для статического сайта:** GitHub Pages или Vercel
- **Для приложения с бэкендом:** Heroku или Railway
- **Для быстрого тестирования:** Vercel (самый простой)

## Настройка домена (опционально)

После деплоя вы можете:
1. Купить домен
2. Настроить CNAME запись
3. Указать кастомный домен в настройках хостинга

## Мониторинг

- GitHub Pages: автоматически показывает статус деплоя
- Vercel: подробная аналитика и логи
- Netlify: функции формы и аналитика
